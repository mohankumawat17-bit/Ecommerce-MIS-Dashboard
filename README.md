# E-commerce MIS Dashboard

This project showcases an interactive MIS dashboard built using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Features
- Total Sales, Orders, Quantity KPIs
- Category-wise Sales Analysis
- Platform-wise Sales Distribution
- Order Status Analysis
- Interactive Filters (Month, Platform, City)

## Project Files
- Dataset (CSV)
- Excel Dashboard
- Dashboard Image

## Insights
- Electronics category generates higher revenue
- Amazon and Flipkart dominate platform sales
- Most orders are successfully delivered
